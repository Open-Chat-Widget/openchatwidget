# Roadmap
This is the roadmap for Open Chat Widget. If you're looking to contribute, please give the roadmap a quick glimpse. We also encourage contributors to help shape the roadmap. Participate in the discussion by joining the [Discord](https://discord.gg/jA4vcJKECy)! 

## Ongoing

- [ ] Create a landing page for Open Chat Widget project. Use Open Chat Widget on the landing page
- [ ] Create docs. Build a Open Chat Widget agent that users can use to ask questions about the docs. 
- [ ] Finish setting up the project. 

## Backlog

- [ ] Add framework-specific backend examples
  Ideas: Express, Hono, Next.js route handlers, and other common Node setups.
- [ ] Add basic theming and branding controls
  Goal: let teams customize the widget without forking the whole component tree.
- [ ] Add better deployment docs
  Goal: explain how to ship the widget and backend in production without extra platform assumptions.
